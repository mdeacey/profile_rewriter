Version History & Roadmap

Current Version (v1.0)

- Prompt customization with options for dialect, formality and tone
- Output length control
- Ability to include or exclude greetings
- Multi-response support
- Bookmarking with URL parameters
- Auto-fill from cookies
- Productivity features (keyboard shortcuts, character count and auto-highlighting)

Planned Features for v2.0+

- Enhanced customization for gender tone, humor, and emotional depth
- Emulate famous personas or create unique character styles
- Responses acknowledge relationship contexts
- Specific jargon and vocabularies for popular industries
- Control content complexity, pacing, and storytelling style
- Ability to specify keywords to include in responses
- Auto-detect settings based on input text
- Compare responses in a dedicated comparison area
- API integration for third-party applications
- Support for non-English languages, incl slang and cultural sensitivities
- Improved error handling with more detailed messages
